Very basic frontend display for [froakbot](https://github.com/hikemalliday/froakbot)

###### Docker Hub:
https://hub.docker.com/r/hikemalliday/froakbot-website-frontend

This website displays all the output messages from 'froakbot', which are fetched from  [froakbot-website-backend](https://github.com/hikemalliday/froakbot-website-backend). They are displayed newest to oldest, fetched in 10 post 'pages' with endless scroll.
